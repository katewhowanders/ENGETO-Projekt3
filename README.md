# ENGETO-Projekt3
# Komplexní webové stránky

## Jednostránkový web obsahuje:
- **Responzivní menu**
- **Ukázka formuláře** - včetně kontroly shody hesla
- **Dark a light mode**
- **Tlačítko pro vyjetí nahoru**
- **Galerie** - obrázky se plynuje zvětšují a zmenšují po najetí kurzorem myši

## Použité technoligie:
- HTML
- CSS
- Javascript
- Ikony pro UI prvky
